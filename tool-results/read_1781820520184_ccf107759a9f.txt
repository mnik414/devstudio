     1‚Üí# DevStudio ‚Äî Web Development Agency Portfolio Platform
     2‚Üí
     3‚Üí## Project Overview
     4‚ÜíA premium web development agency portfolio platform. Originally requested as Laravel/PHP/Filament, delivered as the equivalent Next.js 16 + TypeScript + Prisma + shadcn/ui stack (per environment constraints). All requested features are implemented: Home, Portfolio (+detail), Case Studies (+detail), Blog (+detail), Contact, Estimate calculator, About, Admin panel, SEO (sitemap, robots, structured data), dark mode.
     5‚Üí
     6‚Üí## Tech Stack
     7‚Üí- Next.js 16 (App Router) + TypeScript
     8‚Üí- Prisma ORM + SQLite
     9‚Üí- Tailwind CSS 4 + shadcn/ui (New York)
    10‚Üí- Framer Motion (animations)
    11‚Üí- Zustand (view navigation) + TanStack Query (server state)
    12‚Üí- next-themes (dark mode)
    13‚Üí
    14‚Üí## Color System
    15‚Üí- Primary: #2563EB (blue)
    16‚Üí- Secondary: #0F172A (slate)
    17‚Üí- Accent: #14B8A6 (teal)
    18‚Üí- Background: white (light) / deep slate (dark)
    19‚Üí
    20‚Üí## Architecture (single-route SPA)
    21‚Üí- All content rendered on `/` (per environment rule)
    22‚Üí- Zustand `useNav` store switches between views: home | portfolio | case-studies | blog | contact | estimate | about | admin
    23‚Üí- Detail views (portfolio/blog/case-study) triggered via `openDetail(view, slug)`
    24‚Üí- API routes under `/api/*` for all data + admin CRUD
    25‚Üí
    26‚Üí## Database (Prisma models)
    27‚ÜíPortfolio, PortfolioCategory, Technology, BlogPost, BlogCategory, Tag, CaseStudy, Testimonial, TeamMember, Service, Lead, ContactRequest, Faq, Setting. Seeded via `src/lib/seed.ts`.
    28‚Üí
    29‚Üí## Key Files
    30‚Üí- `src/app/page.tsx` ‚Äî SPA shell (view switcher)
    31‚Üí- `src/lib/store.ts` ‚Äî navigation state
    32‚Üí- `src/lib/hooks.ts` ‚Äî data fetching hooks (TanStack Query)
    33‚Üí- `src/components/site/*` ‚Äî navbar, footer, reveal, counter, section-heading, logo, theme-toggle, scroll-progress
    34‚Üí- `src/components/views/*` ‚Äî all view components
    35‚Üí- `src/app/api/*` ‚Äî API routes (portfolio, blog, case-studies, site, contact, estimate, admin, filters, blog-filters)
    36‚Üí
    37‚Üí## API Routes
    38‚Üí- GET /api/site ‚Äî testimonials, team, services, faqs, settings
    39‚Üí- GET /api/portfolio?featured&category&tech&q&sort&limit
    40‚Üí- GET /api/portfolio/[slug] ‚Äî detail + related + view increment
    41‚Üí- GET /api/filters ‚Äî portfolio categories + technologies
    42‚Üí- GET /api/blog?category&tag&q&featured&limit
    43‚Üí- GET /api/blog/[slug] ‚Äî detail + related
    44‚Üí- GET /api/blog-filters ‚Äî blog categories + tags
    45‚Üí- GET /api/case-studies?featured
    46‚Üí- GET /api/case-studies/[slug]
    47‚Üí- POST /api/contact ‚Äî store contact request (validated)
    48‚Üí- POST /api/estimate ‚Äî calculate estimate + store lead
    49‚Üí- GET/POST/PATCH/DELETE /api/admin?model=... ‚Äî CMS CRUD (X-Admin-Token auth)
